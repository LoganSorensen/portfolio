Conway's Game of Life

Conway's Game of Life is a cellular automaton created in 1970 by John Horton Conway. Each square, or "cell", has two states: alive and dead. The current state of each cell is affected by the state of it's neighbors, and by adhering to a few simple rules complex patterns can emerge as the simulation plays out. I wrote this simulation using React in under a week, following these rules, as an excercise in computer science algorithms.

CRWN Clothing

CRWN Clothing is an E-commerce app built using React. This app handles routing between pages using React Router DOM, interfaces with Firebase to handle user authentication, and uses Redux to manage global state. Additionally, I added Stripe integration to process mock payments, to make this a fully functional store.

Silent Auction Backend

I created this backend for an online auction application I was building with several React developers. Using Node.js, I built a RESTful API for the frontend to interface with. I created a SQLite3 database, which stored information about auctions, bids, and users of the application. Additionally, I implemented user authentication by generating a JWT, and utilized bcrypt.js to hash the user's password. Finally, I added Stripe integration on both the frontend and backend of the application.
